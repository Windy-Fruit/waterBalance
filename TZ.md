# Сервис для студентов с целью расчёта и анализа индивидуального водного баланса

## 1. Цель проекта:

Разработка веб-сервиса, которое помогает студентам рассчитать свою суточную норму воды и отслеживать, сколько воды они выпивают в течение дня и недели.

Сервис предназначен для удобного учёта потребления воды и наглядного отображения результатов.

## 2. Целевая аудитория и роли пользователей

Целевая аудитория: студенты 18–30 лет

Роли пользователей:

Гость:
Может просматривать описание сервиса и пример расчёта нормы воды.

Авторизованный пользователь: 
Может создать личный профиль, получить расчёт суточной нормы воды, добавлять данные о выпитой воде и просматривать статистику.

Администратор:
Имеет доступ к панели администратора Django для управления пользователями и данными.

## 3. Модели данных (Сущности)

### 1. UserProfile (Профиль пользователя): Хранит персональные данные, необходимые для расчёта нормы воды.

Поля: user, age, weight, height,
activity_level, climate

### 2. DailyWaterNorm (Суточная норма воды): Хранит рассчитанное значение суточной нормы воды для пользователя.

Поля: user, calculated_norm, date_created 

### 3. WaterIntake (Приём воды): Модель для хранения информации о выпитой воде.

Поля: user, amount, intake_time, date

## 4. Ключевой функционал (User Stories)

- Регистрация и авторизация пользователей.
- Заполнение профиля с основными параметрами.
- Автоматический расчёт суточной нормы воды.
- Добавление записей о выпитой воде через форму.
- Отображение прогресса выполнения дневной нормы.
- Просмотр статистики потребления воды за неделю.

## 5. Аналитика и обработка данных

Для обработки данных о потреблении воды планируется использование библиотеки Pandas.  

На основе данных будут рассчитываться: общее количество выпитой воды за день, среднее значение за неделю.

Для визуализации результатов будут использоваться графики, построенные с помощью Matplotlib или Plotly.

## 6. Технический стек

- Backend: Python 3.x, Django 4.x  
- Database: SQLite  
- Frontend: HTML, Bootstrap 5  
- Аналитика: Pandas, Matplotlib / Plotly
